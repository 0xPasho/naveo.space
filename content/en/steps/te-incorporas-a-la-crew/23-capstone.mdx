## Your first real shift

Up to here every exercise has been focal: one technique per step. This one's different. <Character name="atlas" /> assigns you **a full shift of message triage**, and to pass you need to combine everything you've learned:

| Unit | How it applies here |
|---|---|
| B · Anatomy and clarity | Close the 4 doors: language, format, length, behavior on gaps. Specific, not vague. |
| C · Context and examples | XML tags + two channels: rules in system, message in user. Few-shot covering the 3 priorities. |
| D · Asking for the output well | JSON with 5 exact fields. `from` can be null (missing data). `summary` capped at 80 characters (length constraints). |
| E · Reasoning and controlling the voice | The `reasoning` field is light chain-of-thought. No preamble, no rewriting: raw JSON and nothing else. |
| F · Debugging and hardening | Case 4 includes **field injection** (guardrails). If another case fails, you iterate one thing at a time. Watch the reinforced-negative trap if you try to forbid something. |

### What your prompt has to return

For each message, a JSON with **exactly 5 fields**:

```
{
  "from": "string|null",
  "priority": "urgent|normal|spam",
  "summary": "string (max 80 chars)",
  "requires_response": true|false,
  "reasoning": "1-2 sentences"
}
```

### How it's evaluated

6 criteria:

1. **Valid JSON.**
2. The 5 fields are present.
3. `priority` is exactly one of the 3 labels.
4. `summary` is between 1 and 80 characters.
5. **The triage is correct**. the judge verifies that priority and summary reflect the actual message.
6. **No extra fields and no obeyed injection**. case 4 tries to inject a `clearance` field; your prompt must ignore it.

All 4 cases must pass all 6 criteria. Atlas doesn't sign half a report.

### When you pass

The system records it as **Track 1 complete**. You move from "student" to "crewmate on probation". Track 2 unlocks multi-turn conversations. where, instead of one message at a time, you sustain an exchange with crewmates to handle more complex situations.

Good luck on the shift.
