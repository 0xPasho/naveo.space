## Una decisión, varias condiciones

Hasta acá las condiciones eran simples: una sola pregunta sí/no. La realidad es que la mayoría de las decisiones de la nave **combinan** dos o más chequeos.

Tres operadores manejan las combinaciones:

| Operador | Significa | Es verdadero cuando |
| --- | --- | --- |
| `Y` (AND) | ambas | las dos partes son verdaderas |
| `O` (OR) | cualquiera | al menos una parte es verdadera |
| `NO` (NOT) | invierte | la parte original es falsa |

### Y: las dos tienen que estar verdes

```
SI esclusa cerrada Y combustible > 50%:
  encender motores
SINO:
  esperar
```

Para encender, **ambas** condiciones tienen que ser verdaderas. Si la esclusa está abierta no encendés, importa poco cuánto combustible tengas. Si tenés 30% no encendés, importa poco que la esclusa esté cerrada.

### O: con una sola alcanza

```
SI puerta forzada O sensor de movimiento activado:
  alertar seguridad
SINO:
  silencio
```

Para alertar, **cualquiera** de las dos alcanza. Es un sistema más sensible, pensado para *"no quiero que se me escape ningún caso"*.

### NO: invertir una condición

```
SI NO crew_a_bordo:
  abortar despegue
```

Se lee *"si NO hay crew a bordo, abortar"*. `NO` toma una condición verdadera y la convierte en falsa (y viceversa).

### El bug más común: confundir Y con O

`Y` y `O` se parecen tres letras pero **arman comportamientos opuestos**.

- `puerta cerrada Y alarma activa`: muy específico. Tiene que pasar todo.
- `puerta cerrada O alarma activa`: muy laxo. Alcanza con que pase algo.

Si confundís uno por otro te pasa una de dos cosas: o el sistema **nunca dispara** (esperabas `O` y pusiste `Y`), o **dispara todo el tiempo** (esperabas `Y` y pusiste `O`). Las dos joden por igual.

> Cuando armes un prompt vas a escribir esto en español: *"si el bio tiene email **y** teléfono, devolvelos. Si **solo** tiene email, devolvé el email. Si **no** tiene ninguno, devolvé null."* Mismo patrón, mismos operadores.

A la derecha, una regla compuesta y cuatro chequeos. ¿En cuál opción la nave acertó qué hacer en cada uno?
